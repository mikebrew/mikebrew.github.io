# Markdown

## Headers

```bash
# h1

## h2

### h3

#### h4

##### h5

###### h6

# Header 1

## Header 2
```

## Emphasis

```bash
*italic*
_italic_

**bold**
__bold__

***bold italic***
**_bold italic_**

~~strikethrough~~

`code`
```

## Lists

```bash
- Item 1
- Item 2

* Item 1
* Item 2

* [ ] Checkbox off
* [x] Checkbox on

1. Item 1
2. Item 2
```

```bash

```

