[jekyllrb.com](http://jekyllrb.com/)

##Create new blog
To install in a new directory  
`jekyll new myblog`  
or install in current directory  
`jekyll new .`

##Generate development server
`jekyll serve`

##Build environment
Current folder will be generated into ./sites  
`jekyll build`  

Generate and watch for changes  
`jekyll build --watch`
